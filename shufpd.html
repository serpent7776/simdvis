SHUFPD / VSHUFPD SSE2 128-bit XMM

Shuffle packed doubles — per 128-bit lane: dst[L·2+0] ← src1[L·2 + bit(2L)], dst[L·2+1] ← src2[L·2 + bit(2L+1)]. Each output qword consumes its own imm8 bit.

imm8 left-click: toggle bit  ·  right-click: edit full imm8  ·  one bit per output qword
